## 🚀 Quick Start

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Configure API keys
cp .env.example .env
nano .env  # Add your Binance API keys

# 3. Test connection
python main.py --test

# 4. Start trading
python main.py
```
